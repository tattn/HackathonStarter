Hackathon starter kit for iOS.

## Installation

```bash
$ git clone https://github.com/tattn/HackathonStarter hack
$ cd hack
$ ./tool/setup.sh
```

Edit Cartfile, and run the following:

```bash
$ carthage bootstrap --platform iOS --cache-builds
```
